# Getting Started

## Overview

Viteshot (named after [Vite](https://vitejs.dev), pronounced `/vit/`) is a command-line tool that renders screenshots from components.

You can learn more about the rationale behind Viteshot in the [Why Viteshot](/docs/why) section.

## Installation

With NPM:

```sh
$ npm install viteshot playwright
```

With Yarn:

```sh
$ yarn add viteshot playwright
```

With PNPM:

```sh
$ pnpm add viteshot playwright
```

## First Time Setup

To set up Viteshot in your project, run:

```sh
$ viteshot init
```

Viteshow will automatically detect which framework you're using and set up the appropriate defaults for you.

## Generate screenshots

To generate screenshots, run:

```sh
$ viteshot
```

You should see an output similar to:

```txt
Capturing: src/components/__screenshots__/darwin/example-MyScreenshot.png
All done.
```

That's it!
